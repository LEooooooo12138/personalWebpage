📋 Current Strengths

  • Clean glassmorphism aesthetic with soft blurs and subtle borders
  • Responsive 12-column Bento grid provides flexible card layout
  • Interactive elements: typewriter effect, terminal mode, clap buttons, AI chat
  • Three.js hero scene adds 3D visual interest

⚠️ Areas for Improvement

  • No dark mode — only light theme, misses an important user preference
  • Colors are ad-hoc hex values, not a systematic token system
  • Card hover effect is subtle (2px lift) — could feel more premium
  • Typography lacks a defined scale & hierarchy
  • Spacing is inconsistent — mixed use of rem, px, and arbitrary values
  • No loading skeletons or empty states defined
  • Focus states and accessibility could be improved

8.2 Skills Page — Terminal Mode Polish

🎯 Proposed Improvements

  • Terminal gets proper syntax highlighting (green prompt, blue dirs, white text)
  • Add tab autocomplete suggestions dropdown
  • Command history with ↑↓ arrow key navigation
  • ASCII art banner on whoami
  • Skill pills get category-colored borders (blue=frontend, green=backend, etc.)

8.3 Experience Page — Timeline Enhancement

🎯 Proposed Improvements

  • Add company/school logos next to each timeline entry
  • Alternating left/right timeline for desktop (more visual interest)
  • Animated timeline line that "draws" on scroll
  • Duration badges (e.g., "2 years", "4 months")
